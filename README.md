# burn
A web platform built with Vue for creating and sending self destructing messages to other people

### Features
-[x] Sending text messages
-[ ] Message Encryption
-[ ] Self Destruction after read
-[ ] Self Destruction after a period of time

### Development
Run mongo
```bash
mongod
```

Run client
```bash
cd client
npm run serve
```

Run server
```bash
cd server
npm run start
```
